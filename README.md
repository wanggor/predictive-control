# Predictive control simulator

Simulator for predictive control system using python.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install package/dependencies.

```bash
pip install -r requirements.txt
```

## Usage

```
python main.py

or 

python3 main.py
```
